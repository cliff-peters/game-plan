# Game Plan

Meeting agenda builder for Directors of Rehabilitation. Generate structured agendas for team meetings and one-on-ones in seconds.

Live: [https://cliff-peters.github.io/game-plan](https://cliff-peters.github.io/game-plan)

## Tabs

**Team Meeting** — Weekly or monthly toggle with topics covering caseload, productivity, scheduling, admissions, discharges, compliance, billing, and more.

**One-on-One** — Employee name and date with topics for performance, caseload, goals, feedback, and compliance. Includes signature lines in the output.

## Features

- Check a topic to expand it and add bullet points underneath
- Add custom topics on the fly
- Output is editable after generating
- Copy to clipboard or print
- No data stored or transmitted — runs entirely in the browser
- Single HTML file, no dependencies
