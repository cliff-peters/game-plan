# Game Plan

**Part of The DOR Suite**

A meeting agenda builder for the directors of rehabilitation. Generate structured, outline-format agendas for team meetings and one-on-ones in seconds.

## What It Does

Game Plan lets you select topics, add bullet points under each one, and generate a clean agenda ready to copy into Google Chat or print for the meeting. No more typing the same agenda format from scratch every week.

## Tabs

**Team Meeting** — Weekly or monthly toggle with a combined topic list covering caseload, productivity, scheduling, admissions, discharges, compliance, billing, and more. Recognition and Shout-Outs always appears last as the closer.

**One-on-One** — Employee name and date with topics for performance, caseload, goals, feedback, and compliance. Includes employee and supervisor signature lines in the output.

## Features

- Check a topic to expand it and add bullet points underneath
- Add custom topics on the fly
- Weekly/monthly toggle changes the output header without switching tabs
- Output is editable after generating so you can make changes on the spot
- Copy to clipboard or print a clean version for the meeting
- No data is stored or transmitted — everything runs locally in the browser
- Single HTML file with no dependencies

## How to Use

1. Open `index.html` in any browser
2. Select Team Meeting or One-on-One
3. Check the topics you want to cover
4. Add bullet points under each topic
5. Click "Generate Plan"
6. Edit the output if needed, then copy or print
